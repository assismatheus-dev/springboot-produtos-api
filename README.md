# API de Produtos - Spring Boot

## Tecnologias
- Java
- Spring Boot
- PostgreSQL
- Swagger

## Funcionalidades
- CRUD completo de produtos
- DTO + validação (@Valid)
- Atualização parcial (PATCH)
- Tratamento de exceções

## 🔗 Endpoints
/api/v1/produtos

## Como rodar

1. Clonar o projeto
2. Configurar PostgreSQL
3. Rodar aplicação

## Swagger
http://localhost:8080/api/swagger-ui/index.html
